# AI Interview Preparation Platform

An AI-powered interview preparation platform built using React and Vite.

## Features

- User Login
- Dashboard
- Interview Questions Section
- Progress Tracking
- Mock Interview Module

## Tech Stack

- React
- Vite
- JavaScript
- CSS

## Installation

git clone https://github.com/adityakhare-18/ai-interview-preparation-platform.git

cd ai-interview-preparation-platform

npm install

npm run dev
