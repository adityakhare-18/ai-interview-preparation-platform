# AI Interview Preparation Platform

## About the Project

I built this project to help students and job seekers practice technical interviews in a more interactive way.

The platform allows users to create an account, attempt interview questions, receive AI-generated feedback, track their progress, and review their previous interview attempts.

The goal of this project was to learn full-stack development while integrating modern AI tools into a real-world application.



## Features

* User Signup and Login
* Secure Authentication using Firebase
* AI Feedback using Google Gemini
* Mock Interview Practice
* Interview History Tracking
* Progress Dashboard
* Persistent Login Sessions
* Cloud Storage with Firestore
* Responsive User Interface



## Technologies Used

* React.js
* JavaScript
* CSS
* Vite
* Firebase Authentication
* Firebase Firestore
* Google Gemini API
* Vercel

## Running Locally

Clone the repository:

```bash
git clone https://github.com/adityakhare-18/ai-interview-preparation-platform.git
```

Move into the project folder:

```bash
cd ai-interview-preparation-platform
```

Install dependencies:

```bash
npm install
```

Create a `.env` file and add your Gemini API key:

```env
VITE_GEMINI_API_KEY=your_api_key
```

Start the development server:

```bash
npm run dev
```
